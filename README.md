# Chess Typescript PIXI

## Installation
Use the package manager [npm](https://www.npmjs.com/) to install chess.

```bash
git clone https://github.com/uliadib12/chess
cd chess
npm install
```

## Build
build static file
```bash
npm run build
```

## Start Server
```bash
npm run start
```
run [localhost:3000](http://localhost:3000/)

## Start Server Development
```bash
npm run start:dev
```
run [localhost:3000](http://localhost:3000/)
